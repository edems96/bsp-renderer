Introduction

bsp-renderer is a program for displaying levels stored in the quake3 bsp map format. It allows the user to move around in the levels using and first person camera and collision detection. The purpose of this document is to give an overview of the bsp-renderer application, including how to use it, which third party libraries and thrid party code it uses and some technical information. The downloadable distribution of the program contains the source code and prebuilt executables. It also contains the libraries necessary to build the source, and a Quake III level called "The Forlorn Hope", which was found at http://www.users.on.net/justonefix/.

bsp-renderer is released under GPL version 3. However, it relies on several third party libraries, as well as pieces or source code from other open source programs. An overview of these licenses is given here.

Source code licenses and credits

The source for the collision detection (but not the collision response) is based on code from http://www.devmaster.net/articles/quake3collision/, which in turn comes from the Quake III source.
The tessellation of the Bezier patches is based on code from http://graphics.cs.brown.edu/games/quake/quake3.html. The code comes originally from The Octagon Engine by Paul Baker, http://www.paulsprojects.net/opengl/octagon/octagon.html
